## 这里是web版本的部署，基于Flask

# 部署指南

## 安装 Flask

如果您还没有安装 Flask，您可以使用 pip 来安装它。在您的终端或命令提示符中运行以下命令：

```bash
pip install flask
```

## 保存代码

将 app.py和index.html上传至服务器（或下载至本地）

## 运行 Flask 应用

在包含 app.py 的目录中，运行以下命令来启动 Flask 应用：

```bash
python3 app.py
```

## 访问应用

在浏览器中打开 http://127.0.0.1:5000 。您应该能看到您的前端页面，并能够提交表单以获取 tokens。
